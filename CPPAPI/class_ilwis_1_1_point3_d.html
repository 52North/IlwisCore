Detailed Description

template<typename CrdType = qint32>
class Ilwis::Point3D< CrdType >

A point in 3D space. It is a derivative of the Point class. As the base class already offer the x,y components no further implementation of these is needed. The class mostely implements special operations on 3D points and a z coordinate
