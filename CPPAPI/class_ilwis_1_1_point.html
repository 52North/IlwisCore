Detailed Description

template<class CrdType = qint32>
class Ilwis::Point< CrdType >

A representation of a point in the cartesian space. It is the base class for the Point2D and Point3D classes. The type of the coordinate (CrdType) must support the ==,+, -, * ,/ and copy operations (usually numerical) for this class to work properly. each point has a x and a y property. The meaning of the point is determined by the coordinate space it is used in. On purpose this meaning is now left upto the using context ( realworld coordinate systems, grid system, uni-sphere systems etc...).
